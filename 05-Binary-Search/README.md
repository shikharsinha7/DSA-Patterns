# Binary Search

**When to use:** Sorted array (even if rotated), and the search space can be halved each time - look out for phrasing like 'minimum/maximum value satisfying some condition'.

The base template stays the same every time, what changes is figuring out the condition for which half to throw away. That's the actual hard part, not the binary search itself.
