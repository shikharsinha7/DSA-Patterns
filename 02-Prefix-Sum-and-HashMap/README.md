# Prefix Sum + HashMap

**When to use:** When the question is about subarray sum equal to K, or there are negative numbers in the array so sliding window won't work, or you need to count subarrays satisfying some condition.

Keep mixing this up with plain sliding window at first, the giveaway is negative numbers - if negatives are allowed sliding window breaks, that's when prefix sum + hashmap comes in.
