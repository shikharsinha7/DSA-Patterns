# Two Pointers

**When to use:** Sorted array, looking for pairs/triplets, checking palindromes, or anything where you're comparing stuff from both ends moving inward.

Difference from sliding window that I keep forgetting: two pointers usually move toward each other or independently, sliding window pointers move together expanding/shrinking one continuous range.
