# Backtracking

**When to use:** Generating all combinations/permutations/subsets, or any constraint satisfaction problem (like N-Queens).

General template I keep reusing for all of these: make a choice, recurse, undo the choice (backtrack) before trying the next option. Once you see this shape a few times it clicks.
