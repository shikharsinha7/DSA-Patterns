# Heap / Priority Queue / Top K

**When to use:** Kth largest/smallest, median of a stream, task scheduling, K closest points - anywhere you need the 'top/bottom K' instead of fully sorting everything.

Two heaps (max heap + min heap together) shows up specifically for median-of-stream type problems. Rest of these are usually single heap of size K.
