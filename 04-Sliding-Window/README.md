# Sliding Window

**When to use:** Subarray or substring problems, fixed size or variable size window, usually with all positive numbers and contiguous elements.

Get confused on when to expand vs shrink, so noting it here: expand by moving right pointer, shrink by moving left pointer once window becomes invalid (or once you've got a valid answer and want to try shrinking further).
